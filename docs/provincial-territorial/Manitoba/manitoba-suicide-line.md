## About
The Manitoba Suicide Prevention & Support Line is a **toll-free, confidential 24-hour** crisis line for people:
- thinking about suicide
- worried about someone else
- affected by a suicide loss

It is run by trained crisis counsellors from Klinic Community Health.

## Coverage
**Manitoba-wide** (toll-free calling).

## Contact
- **Call (24/7):** **1-877-435-7170**
- **Website:** https://reasontolive.ca/

## Hours
- **24/7/365**

## Languages
- Translation is available **in more than 175 languages** upon request.
